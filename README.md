## HOWTO: 

Template:
```shell
java -Xmx500M -cp <ANTLR JAR> org.antlr.v4.Tool -Dlanguage=Python3 <GRAMMAR>
```

Example:
```shell
java -Xmx500M -cp /usr/local/lib/antlr-4.8-complete.jar org.antlr.v4.Tool -Dlanguage=Python3 Hello.g4
```
